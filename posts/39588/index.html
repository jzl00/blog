发表于 2026-07-16 分类于 前端开发 阅读次数：

一、关于 React

  • 英文官网：https://reactjs.org/
  • 中文官网：https://react.docschina.org/

React 起源于 Facebook 的内部项目，因为该公司对市场上所有 JavaScript MVC 框架都不满意，就决定自己写一套，用来架设 Instagram 的网站。做出来以后发现这套东西很好用，就在 2013 年 5 月开源了。

React 在发展过程中一直跟随原生 JS 的脚步，特别是从 v16.0 版本开始使用 class 来创建组件。

2015 年推出了使用 React 编写移动端 App —— React Native。

重要版本发布时间

序号 版本号 发布时间 重要更新
1 16 2017年9月26日 引入 ES6 类组件

3 16.4 2018年5月23日 生命周期更新
4 16.8 2019年2月6日 引入 React Hooks 函数组件
5 17.0 2020年10月20日 过渡版本
6 18.0 2022年3月29日 写法改变，严格模式改变

三、React 基础

元素与组件

  • 元素：组件的最小单位，单个 HTML 标签
  • 组件：由多个元素嵌套组合而成的自定义标签

函数式组件

  1. 组件是一个函数
  2. 组件名称首字母要大写
  3. 函数内部要 return 出模板

HTML 共有 107 个标签，如果首字母小写会先对比 107 次；大写直接当作自定义组件。

类组件

  1. 定义 class 继承 Component
  2. 必须实现 render 钩子函数
  3. render 中 return 组件模板

JSX 语法

函数组件或类组件中 return 的模板就是 JSX，React 会将其解析为虚拟 DOM，再处理成真实 DOM。

  • 遇到 < 理解为元素或组件
  • 遇到 { 理解为 JS 表达式
