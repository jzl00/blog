Vue3 基础教程（选项式 API）

发表于 2026-07-16 分类于 前端开发 阅读次数：

Vue 介绍

渐进式框架：可以用最简单的方式直接导入 vue.js 使用，随着项目变大，可以逐步集成周边插件。

  • 版本：Vue 3
  • 学习方式：选项式 API
  • 官网：https://cn.vuejs.org/
  • CDN 地址：https://unpkg.com/vue@3/dist/vue.global.js

Vue 接入步骤

  1. 下载 vue.js 文件
  2. 在 html 中引入
  3. 创建 Vue 实例
  4. 挂载到页面元素

模板语法

双大括号插值

{{ 表达式 }} - 可以写 JS 表达式，能取到 data 中的数据

常用指令

指令 作用 简写
v-text 渲染纯文本，不解析标签 -
v-html 渲染 HTML，解析标签 -

动态属性：:[attr]="url" 可以动态绑定不同属性

响应式原理

  1. 虚拟 DOM 本质是一个 JS 对象
  2. 数据改变先更新虚拟 DOM，不直接操作真实 DOM
  3. 通过 DIFF 算法 比较新旧虚拟 DOM，只更新变化的部分
  4. nextTick 在真实 DOM 更新后调用

data 必须是函数返回对象：保证每个组件有独立的数据作用域，通过 Proxy 实现响应式

计算属性 (computed)

特点

  1. ✅ 有缓存
  2. ✅ 立即执行一次
  3. ✅ 监听多个依赖，任一改变就重新计算
  4. ✅ 必须有 return 返回值
  5. ✅ 多个值影响一个值

可写计算属性

侦听器 (watch)

特点

  1. ✅ 惰性：默认不立即执行
  2. ✅ 无需返回值
  3. ✅ 支持异步操作
  4. ✅ 没有缓存
  5. ✅ 监听一个值，影响多个值

watch vs computed 对比

特性 computed watch
立即执行 ✅ 是 ❌ 否（惰性）
返回值 ✅ 需要 ❌ 不需要
异步 ❌ 不支持 ✅ 支持
缓存 ✅ 有 ❌ 无
监听数量 多个值 一个值
场景 多对一 一对多

Class 和 Style 绑定

Class 绑定

  • 字符串：:class="className"
  • 对象：:class="{ active: isActive }" 布尔值控制
  • 数组：:class="[classA, classB]"

绑定的 class 会和原有的 class 合并

Style 绑定

同样支持字符串、对象、数组三种方式

条件渲染

v-if / v-else-if / v-else

  • 条件成立才渲染，不成立直接移除
  • 具有惰性
  • 适合切换不频繁的场景

v-show

  • 始终渲染，通过 CSS display 控制显示隐藏
  • 非惰性
  • 适合频繁切换的场景

列表渲染 v-for

可遍历类型

  • 数字：v-for="num in 10" 从 1 开始
  • 字符串：v-for="str in '你好'" 逐个字符
  • 数组：v-for="(item, index) in arr"
  • 对象：v-for="(value, key, index) in obj"

key 属性

使用 v-for 必须加 key，且唯一。帮助虚拟 DOM 正确识别元素位置，提升 diff 性能。

v-if 和 v-for 优先级

  • Vue 2：for 优先级高于 if
  • Vue 3：if 优先级高于 for（条件不成立不循环）

不推荐一起使用

两种方式

单向数据流
