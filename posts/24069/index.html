上拉加载更多

  • hasMore — 是否还有更多数据
  • loadMore — 上拉触发的回调函数
  • 默认执行一次，用 count 标记页码（默认 1），每次请求后 count + 1

下拉刷新

  • onRefresh — 下拉后执行
  • 刷新成功后将 count 重置为 2，为下次上拉做准备

回到顶部

实现思路：

  1. 创建 contentRef 标记内容区域
  2. 监听滚动事件，记录滚动距离
  3. 滚动距离大于 250px 时显示回到顶部按钮
  4. 点击后设置滚动距离为 0

实现流程

  1. 创建登录请求接口
  2. 创建 saga.js 异步操作
  3. 创建 reducer.js 全局状态
  4. types 增加请求方法和修改方法名
  5. store/reducer.js 导出状态模块
  6. store/index.js 配置模块
  7. mySaga 中配置异步事件监听
  8. login.jsx 中调用异步事件请求
  9. 请求成功后跳转到首页
