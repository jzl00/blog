WXSS 样式

尺寸单位 rpx

rpx（responsive pixel） — 根据屏幕宽度自适应，规定屏幕宽为 750rpx。

设备 rpx 换算 px px 换算 rpx
iPhone5 1rpx = 0.42px 1px = 2.34rpx
iPhone6 1rpx = 0.5px 1px = 2rpx
iPhone6 Plus 1rpx = 0.552px 1px = 1.81rpx

建议以 iPhone6 作为视觉稿标准

全局样式与局部样式

  • app.wxss — 全局样式，作用于所有页面
  • 页面 .wxss — 局部样式，仅作用于当前页面，覆盖同名选择器

WXS 脚本语言

WXS（WeiXin Script）是小程序的脚本语言，结合 WXML 构建页面结构。

WXS 与 JavaScript 不同，有自己的语法，不完全一致

包含模块：变量、注释、运算符、语句、数据类型、基础类库

常用组件

基础组件

navigator 页面跳转

open-type 类型：

  • navigate — 入栈跳转（push），有返回按钮
  • redirect — 重定向，关闭当前页打开新页，无返回
  • switchTab — 跳转到 tab 页，关闭其他非 tab 页
  • reLaunch — 关闭所有页面，打开某个页面（可跳 tab）
  • navigateBack — 返回上一页

项目开发

Vant Weapp UI 组件库

文档地址：https://vant-contrib.gitee.io/vant-weapp/

安装步骤：

  1. 初始化 package.json

  2. 安装 Vant

  3. 删除 app.json 中的 "style": "v2"（新样式难以覆盖）

  4. 微信开发者工具 → 工具 → 构建 npm → 勾选「使用 npm 模块」

  5. 导入组件使用，构建后可删除 node_modules

封装数据请求

baseURL 需在小程序后台配置域名白名单，开发阶段可在开发者工具设置「不校验合法域名」

API 封装

在 api/ 文件夹中封装各模块请求方法，页面直接导入调用即可。
